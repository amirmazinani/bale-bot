# 🤖 بات تراز - سیستم کاملاً داینامیک

## 🎯 معرفی
**بات تراز** الان کاملاً داینامیک شده! میتونید:

✅ **صفحات جدید** اضافه کنید - بدون کدنویسی  
✅ **دکمههای جدید** اضافه کنید - فقط JSON ویرایش کنید  
✅ **منوها رو تغییر** بدید - در عرض چند ثانیه  
✅ **محتوا رو آپدیت** کنید - بدون ریاستارت طولانی  

## 🚀 شروع سریع

### قدم ۱: فعال کردن سیستم داینامیک
```bash
python setup_dynamic.py
```

### قدم ۲: اجرای بات
```bash
python main.py
```

### قدم ۳: تست تغییرات
۱. فایل `content.json` رو ویرایش کنید  
۲. بات رو ریاستارت کنید  
۳. تغییرات رو توی چت تست کنید

## ✨ اضافه کردن صفحه جدید (بدون کدنویسی!)

### مثال: اضافه کردن صفحه "تیم ما"

**فقط این رو به `content.json` اضافه کنید:**

```json
{
  "screens": {
    "our_team": {
      "text": "👥 تیم ما\\n\\nاعضای حرفهای تیم ما...",
      "parent": "main_menu",
      "buttons": [
        [
          {
            "text": "📞 تماس",
            "route": "contact"
          }
        ],
        [
          {
            "text": "🏠 منوی اصلی",
            "route": "main_menu"
          }
        ]
      ]
    }
  }
}
```

**و این دکمه رو به منوی اصلی اضافه کنید:**

```json
{
  "text": "👥 تیم ما",
  "route": "our_team"
}
```

**تمام!** صفحه جدید با همه قابلیتها ساخته میشه.

## 🔧 ویرایش محتوا

### ۱. تغییر متن صفحات
```json
{
  "screens": {
    "about": {
      "text": "متن جدید درباره ما...",
      "buttons": [...]
    }
  }
}
```

### ۲. اضافه/حذف دکمه
```json
{
  "main_menu": {
    "buttons": [
      // دکمههای قبلی
      [
        {
          "text": "دکمه جدید",
          "route": "new_page"
        }
      ]
      // دکمه قدیمی رو حذف کنید
    ]
  }
}
```

### ۳. تغییر ریپلای کیبورد
```json
{
  "reply_keyboard": {
    "buttons": [
      {
        "text": "📞 تماس",
        "action": "contact"
      },
      {
        "text": "🏠 خانه",
        "action": "main_menu"
      },
      {
        "text": "📝 وبلاگ",
        "route": "blog"
      }
    ]
  }
}
```

## 🚨 عیبیابی

### مشکل: صفحه نمایش داده نمیشه
✅ **علت**: `screen_key` اشتباه نوشته شده  
✅ **راهحل**: از `screen_key` موجود استفاده کنید

### مشکل: دکمه کار نمیکنه  
✅ **علت**: `route` اشتباه نوشته شده  
✅ **راهحل**: مطمئن شید صفحه در `screens` تعریف شده

### مشکل: بات اجرا نمیشه
✅ **علت**: JSON syntax error  
✅ **راهحل**: از JSON validator استفاده کنید
